Браузери

Текстові документи, що містять розмітку мовою HTML (такі документи зазвичай мають розширення .html або .htm), обробляються спеціальними програмами, які відображають документ у його форматованому вигляді. Такі програми, звані «браузерами» або «інтернет- браузерами», зазвичай надають користувачеві зручний інтерфейс для запиту веб-сторінок, їх перегляду (і виведення на інші зовнішні пристрої) і, при необхідності, надсилання введених користувачем даних на сервер. Найбільш популярними на сьогоднішній день браузерами є Internet Explorer, Mozilla Firefox, Apple Safari, Google Chrome та Opera (див.: Браузер # Ринкові частки).

Internet Explorer

Windows Internet Explorer (читається інтернет експлорер, раніше - Microsoft Internet Explorer або просто Internet Explorer, скорочено MSIE чи IE; /aɪ iː/) — серія браузерів, яку розробляє корпорація Microsoft з 1995 року. Входить у комплект операційні системи сімейства Windows. Займає перше місце за кількістю користувачів (ринкова частка у квітні 2010 року - ▼59,95 %, у січні 2011 — ▼45,99 %, у лютому 2012 – ▲52,84% (за даними GlobalStats)). У Росії, за даними порталу LiveInternet, на березень 2012 посідає третє місце з ▼15,6 % користувачів[3], перше місце у браузера Google Chrome з ▲ 19,5%.

Mozilla Firefox

Mozilla Firefox (вимовляється моузи́ла файрфокс, [moʊˈzɪlə ˈfaɪ(ɹ)fɑks] — браузер, що вільно розповсюджується, що входить в набір програм Mozilla Application Suite, розробкою та розповсюдженням якого займається Mozilla Corporation. Третій за популярністю браузер у світі та другий серед вільного ПЗ — у січні 2012 року його ринкова частка становила ▼24,78 %. Браузер має особливий успіх у деяких країнах, зокрема, у Німеччині та Польщі це найпопулярніший браузер із часткою 55% та 47% відповідно [4]. У Росії Firefox займає перше місце за популярністю серед десктопних браузерів з часткою 30,74% користувачів до липня 2011 року.
